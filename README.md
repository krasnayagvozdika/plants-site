# plants-site

Сайт предприятия "Красная гвоздика" с публичным каталогом растений и PHP-админкой.

## Основное

- Публичный каталог читает данные из `data/catalog.json`.
- Админка находится в `admin/`.
- Серверная логика находится в `backend/`.
- Загруженные через админку изображения сохраняются в `images/catalog/`.
- База данных, npm и composer не нужны.

## Требования

- PHP 8.0 или новее.
- PHP extensions: `json`, `session`, `gd`.
- `gd` обязателен: без него загрузка изображений отключена, чтобы не сохранять на хостинг крупные оригиналы.

## Права на запись

Для рабочей админки хостинг должен разрешать запись в:

- `data/`
- `data/catalog.json`
- `data/sessions/`
- `images/catalog/`
- `backend/` для создания `backend/config.local.php`

## Админка

- URL: `/admin/login.php`
- Логин по умолчанию: `admin`
- Пароль по умолчанию: `admin111`

После размещения пароль нужно сразу изменить через админку.

## Локальный запуск

```bash
php -S localhost:8080
```

После запуска:

- сайт: `http://localhost:8080`
- админка: `http://localhost:8080/admin/login.php`
- API каталога: `http://localhost:8080/api/catalog.php`

## Архив для хостинга

Для передачи хостеру собирается zip без `.git`, Dockerfile, docs и временных session-файлов. В архив кладется `HOSTING_README.txt` с короткой инструкцией по установке.
