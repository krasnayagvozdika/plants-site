Витрина "Красная гвоздика": инструкция для хостинга

Что внутри
- Публичный сайт с каталогом растений.
- PHP-админка для управления каталогом.
- Локальное хранение данных без базы данных.

Требования к хостингу
- PHP 7.4 или новее.
- Обязательные расширения PHP: json, session, gd.
- Должна быть разрешена загрузка файлов через PHP.

Важно: расширение GD обязательно.
Без GD админка не будет принимать изображения, потому что крупные фото должны уменьшаться перед сохранением.

Куда заливать
- Можно залить в корень домена или поддомена.
- Можно залить в подпапку существующего сайта.

Права на запись
Выдать права на запись для:
- data/
- data/catalog.json
- data/sessions/
- images/catalog/
- backend/

Папка backend/ нужна для файла backend/config.local.php, который создается после смены пароля.

Первый вход в админку
- Адрес: /admin/login.php
- Логин: admin
- Пароль: admin111

После первого входа сразу сменить пароль в админке.

Что проверить после заливки
- Открывается главная страница.
- Открывается каталог.
- Работает вход в админку.
- Можно добавить растение.
- Можно загрузить изображение.
- Загруженное изображение появляется в images/catalog/.
- Можно удалить тестовое растение.
- Переключатель "В наличии" работает.
- Скачивается Excel.
- Открывается PDF-каталог.

База данных, npm, composer и сборка проекта не нужны.
