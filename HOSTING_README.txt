Витрина "Красная гвоздика": установка на хостинг

1. Распакуйте содержимое архива в папку сайта.
   Можно размещать в корне домена/поддомена или в подпапке существующего сайта.

2. На хостинге должен работать PHP.
   Желательно PHP 7.4 или новее.
   Обязательные возможности: json, session, загрузка файлов.
   Желательно: gd для уменьшения загружаемых изображений.

3. Дайте права на запись:
   - data/
   - data/catalog.json
   - data/sessions/
   - images/catalog/
   - backend/ для создания backend/config.local.php при смене пароля

4. Откройте админку:
   /admin/login.php

   Логин: admin
   Пароль: admin111

5. Сразу после первого входа смените пароль в админке.

6. После проверки убедитесь, что работают:
   - публичный каталог;
   - вход в админку;
   - добавление и удаление позиции;
   - загрузка изображения;
   - переключатель "В наличии";
   - скачивание Excel;
   - печатный PDF-каталог.

База данных, npm, composer и сборка проекта не нужны.
